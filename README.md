# Akshara 32K Tokenizer UI

This repository contains the premium demonstration UI for the Akshara 32K tokenization model.

## Live Demo
The application is deployed and currently live on AWS Amplify:
**[https://main.d3t63e4qj66hub.amplifyapp.com/](https://main.d3t63e4qj66hub.amplifyapp.com/)**

## Overview
This UI highlights the efficiency of the Akshara 32K tokenization model, specifically tailored for the Telugu language, and visually compares it against industry standards like GPT-4o, Gemini 1.5 Pro, and Sarvam AI.

### Features
- **Live Tokenizer Demo:** Dynamic rendering of generated tokens with staggered entrance animations.
- **Micro-Animations:** Tooltips with token IDs and byte sizes, plus counting animations for stats.
- **Premium Aesthetics:** Dark space theme with dynamic interactive glow, twinkling stars, and glassmorphism elements.
- **Comparison Engine:** Side-by-side metric comparison demonstrating Token Fertility improvements.
